1. Общие положения

Настоящая Политика конфиденциальности описывает порядок обработки данных пользователей мобильного приложения «SOSвидания» (далее — Приложение).

Приложение разработано и принадлежит физическому лицу Наумовичу Сергею Сереевичу (далее — Владелец).

Политика составлена в соответствии с:

  • Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных»
  • Требованиями магазина приложений RuStore

2. Какое приложение

«SOSвидания» — это симулятор входящих звонков. Приложение позволяет создать фейковый входящий вызов с настраиваемыми параметрами: имя контакта, фото, звук звонка, голосовое сообщение.

Приложение НЕ совершает реальные звонки и не подключается к телефонной сети.

3. Какие данные собираем

3.1 Личные данные (не собираются)

Приложение НЕ собирает и НЕ передаёт на серверы следующие данные:

  • Имена и фамилии
  • Номера телефонов
  • Фотографии и изображения
  • Голосовые записи
  • Контакты из телефонной книги
  • Историю звонков

Вся эта информация хранится только локально на вашем устройстве и недоступна владельцу приложения или третьим лицам.

3.2 Технические данные (собираются анонимно)

Для обеспечения стабильной работы приложения могут автоматически собираться следующие обезличенные технические данные:

  • Модель устройства
  • Версия операционной системы
  • Язык системы
  • Версия приложения

    • Эти данные не позволяют идентифицировать пользователя и используются только для:

    • Устранения технических ошибок
    • Улучшения совместимости с устройствами
    • Статистики использования функций

    4. Разрешения приложения

    Приложение может запрашивать следующие разрешения на устройстве:

    Разрешение Зачем нужно Передача данных
    Микрофон Запись голосового сообщения для симуляции звонка ❌ Нет, запись хранится только на устройстве
    Камера / Галерея Выбор фото для контакта ❌ Нет, фото хранится только на устройстве
    Хранилище Сохранение настроек и данных профиля ❌ Нет, данные хранятся только на устройстве
    Контакты Опционально — для выбора имени контакта ❌ Нет, доступ только для чтения, данные не сохраняются
    Уведомления Показ экрана входящего звонка ❌ Нет передачи
    Вибрация Тактильный отклик при звонке ❌ Нет передачи
    Важно

    Все разрешения используются исключительно для локальной работы функций симуляции. Данные не передаются владельцу приложения, не отправляются на серверы и не передаются третьим лицам.

    5. Хранение данных

    Все пользовательские данные хранятся локально на устройстве:

    • Профили контактов (имя, фото)
    • Голосовые записи
    • Настройки приложения

    Данные удаляются при:

    • Удалении приложения
    • Очистке данных в настройках устройства
    • Удалении профиля пользователем

    6. Сторонние сервисы

    Приложение может содержать рекламу от сторонних сетей (Google AdMob, Яндекс Мобильная Реклама). Эти сервисы могут собирать:

    • Рекламный идентификатор (GAID / AAID)
    • IP-адрес (в обезличенном виде)
    • Информацию об устройстве

    Владелец приложения не контролирует сбор данных рекламными сетями. Вы можете отключить персонализированную рекламу в настройках устройства.

    7. Права пользователя (ФЗ-152)

    В соответствии с Федеральным законом № 152-ФЗ вы имеете право:

    • Отозвать согласие на обработку данных — удалите приложение
    • Удалить свои данные — очистите данные приложения в настройках устройства
    • Запретить доступ к разрешениям — отключите в настройках устройства
    • Получить информацию об обработке данных — напишите владельцу

    8. Дети

    Приложение не предназначено для детей младше 13 лет. Владелец не собирает сознательно данные детей младше этого возраста.

    Владелец может обновлять эту Политику конфиденциальности. Актуальная версия всегда доступна в приложении и по ссылке в магазине приложений.
